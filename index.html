財富金三角
回報、安全與時間的抉擇

在理財世界裡，有一個不變的定律：我們無法同時擁有「極高回報」、「絕對安全」和「隨時提款」的完美產品。理財的本質，就是根據您當下的人生階段，選擇最適合的取捨。

1. 定期 / 儲蓄戶口 🏦
安全 + 流動性 ➔ 捨棄回報
客人視角：資金隨時可以調動，本金絕對安全。但代價是利息微薄，長遠來看，購買力會被通脹無聲無息地蠶食。適合存放應急錢，但不適合用來增值財富。
2. 股票 / 股票基金 📈
高回報 + 流動性 ➔ 捨棄安全
客人視角：您可以隨時在市場賣出套現，且有機會賺取數以倍計的回報。代價是必須承受極大的心理壓力與本金虧損風險，屬於「高風險、高回報」的戰場。
3. 儲蓄分紅保險 🌱
安全 + 長線回報 ➔ 捨棄短期流動性
客人視角：利用「時間」作為護城河。在計劃初期交出資金隨時調動的自由，換取免受市場波動驚嚇的安心感，以及長線滾雪球般的複利回報，是財富防守與傳承的基石。
4. 優質債券 📜
穩定現金流 + 中度安全 ➔ 捨棄爆發力
客人視角：將資金借給政府或大型企業一段時間，換取合約承諾的定期派息。回報比定期高，風險比股票低，但缺乏資產倍增的爆發力。
